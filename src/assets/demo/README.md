# Capturas de la demo guiada (/demo)

Poné acá las capturas del tablero. El nombre del archivo = el paso.
Formatos: .png, .webp o .jpg. Se cargan solas por nombre.

| Archivo            | Pantalla a capturar                                   |
|--------------------|-------------------------------------------------------|
| biblioteca.png     | /library — biblioteca de ejercicios con filtros       |
| ficha.png          | Detalle/ficha de un ejercicio (lista para imprimir)   |
| pacientes.png      | /patients — lista de pacientes                         |
| progreso.png       | /patients/:id — panel de progreso clínico             |
| sesion.png         | /sessions/builder — armado de sesión                   |

## IMPORTANTE — Privacidad (Ley 25.326)
Las capturas NO deben mostrar datos de pacientes reales.
Usá datos de ejemplo / ficticios (ej: "Juan Pérez", "María Gómez").

Recomendado: capturas anchas (~1280px) para que se vean nítidas.
